# yadc-si

An SI unit system built on top of [`yadc`](../yadc/) — 117 quantities (Mass, Force, Energy, ThermodynamicTemperature, Length, Velocity, …), organized into 9 feature-gated families:

| Feature | Examples |
|---|---|
| `mechanical` | Length, Mass, Time, Force, Energy, Power, Velocity, Acceleration, Frequency, Torque, … |
| `angular` | Angle, SolidAngle, AngularVelocity, AngularAcceleration, … |
| `electromagnetic` | ElectricCurrent, ElectricPotential, Capacitance, MagneticFlux, ElectricalResistance, … |
| `thermal` | ThermodynamicTemperature, TemperatureInterval, HeatCapacity, ThermalConductivity, … |
| `fluid` | DynamicViscosity, MassRate, VolumeRate, SurfaceTension, DiffusionCoefficient, … |
| `density` | MassDensity, LinearMassDensity, ArealMassDensity, VolumetricNumberDensity, … |
| `chemistry` | AmountOfSubstance, MolarMass, MolarConcentration, Radioactivity, CatalyticActivity, … |
| `photometric` | Luminance, LuminousIntensity, Illuminance, RadiantExposure |
| `information` | Information, InformationRate |

All features are on by default. Disable selectively with `default-features = false` + an explicit feature list to reduce compile time.

## Quick start

```rust
use yadc_si::{mechanical::*, Format};

let mass = Mass::from_unit(70.0, MassUnit::Kilogram);
let accel = Acceleration::from_unit(9.81, AccelerationUnit::MeterPerSecondSquared);
let force: Force = mass * accel;
println!("{}", force.fmt_as_unit(ForceUnit::Newton, Format::Short));
```

See [`yadc/examples/basic_use.rs`](../yadc/examples/basic_use.rs) for a feature tour.

## Optional features

- `serde` (off by default) — derives compact `Dim` string serialization (`"m, kg, s^-2 [Affine]"`) and transparent `Qty<D>` serde via `f64`. Enable with `features = ["serde"]`.

## Origin of the unit data

Conversion factors, symbols, and labels are derived from the [`iliekturtles/uom`](https://github.com/iliekturtles/uom) project (MIT/Apache-2.0). The unit system definition lives in [`yadc-codegen/si.toml`](../yadc-codegen/si.toml); all source files in this crate are generated by [`yadc-codegen`](../yadc-codegen/).
